# 🎯 How to Add Your .lottie File

## 📁 Step 1: Add Your .lottie File
1. **Copy your .lottie file** to the `public/` folder
2. **Name it**: `your-animation.lottie` (or update the src in Login.js)

## ⚙️ Step 2: Update the File Name (if needed)
In `src/components/Login.js`, find this line:
```javascript
src="/your-animation.lottie"
```

Change it to your actual filename:
```javascript
src="/my-login-animation.lottie"
```

2. Update the filename in Login.js if needed
